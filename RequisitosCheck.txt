Tela Visão Geral (Dashboard SPA)

Tela de Cadastro (SPA)

Tela de Visualização (SPA)

✔️ Requisito 4 — Armazenamento de dados

✔️ Armazenamento em MongoDB ou CSV fallback.
✔️ Totalmente válido para o trabalho.
✔️ Você pode explicar ao professor:

“A aplicação usa MongoDB como banco principal e automaticamente cai para CSV caso não haja conexão, garantindo persistência mesmo offline.”

É um ponto extremamente positivo.

✔️ Requisito 5 — Recuperação dos dados cadastrados

✔️ A tela Visualizar Candidaturas mostra todos os registros do banco/CSV.
✔️ Com tabela, scroll, link, tudo certo.

✔️ Requisito 6 — Gráficos com base nos dados cadastrados

✔️ Gráficos atualizam automaticamente.
✔️ Totalmente integrados ao Tkinter.
✔️ Usam Matplotlib conforme pedido no edital.

✔️ O dashboard lê do DataStore.
